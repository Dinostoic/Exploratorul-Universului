    Exploratorul Universului este o aplicație web educațională interactivă dedicată astronomiei și explorării spațiale. Conține 12 lecții enciclopedice, un quiz cu 54 de întrebări, sistem de realizări, jurnal de activitate și multe altele. Proiectul a fost realizat folosind HTML5, CSS3 și JavaScript pur, demonstrând concepte de programare web precum manipularea DOM-ului, localStorage, Canvas API și animații CSS.
    Tehnologii folosite
    HTML5 CSS3 JavaScript ES6+ Canvas API Web Audio API localStorage CSS Animations Responsive Design Font Awesome 6 Google Fonts
